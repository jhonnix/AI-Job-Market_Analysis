# Excel Sales Dashboard

## Project Overview

This project analyzes sales performance using Microsoft Excel.

## Tools Used

- Microsoft Excel 2019
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- VLOOKUP

## Key Metrics

- Total Sales
- Monthly Revenue
- Top Products
- Regional Performance
